# 15_midnighters

###Prerequisites:

Run in console `pip install -r requirements.txt` to install 3rd party modules.

---

This module display all code-addictive programmers from [devman.org](https://devman.org)
Obviously midnight starts from 12:00pm.

###How to use:

You can edit in-script constants to get result as you want.

| Constant | Description |
| --- | --- |
| `MIDNIGHT_END` | determines who should be considered for a 'code-addictive' |
